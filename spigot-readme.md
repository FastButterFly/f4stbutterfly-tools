**F4stbutterfly-tools**

Fairly basic plugin providing following functionality

- Basic
- - Change join and quit messages
- Admin commands:
- - /gamemode mode target - Changes gamemode
- - /clear target - Clears inventory
- - /enchant target enchantment level - Enchants item in hand
- - /ec target - Opens enderchest
- - /feed target - Feed target
- - /fly target - Change fly mode
- - /give target item amount - Give item
- - /god target - Change god mode
- - /heal target - Heal
- - /helpop message - Send message to server admins
- - /invsee target - See inventory
- - /kill target - Kill
- - /repair - Repair item in hand

More features are going to be added later!
